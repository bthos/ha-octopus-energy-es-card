# Octopus Energy España Consumption Card

Lovelace card for displaying consumption graphs and tariff comparisons for Octopus Energy España integration.

## Features

- Consumption visualization for day/week/month periods
- Period navigation with Previous/Next buttons
- Tariff comparison with cost breakdown
- Period breakdown (P1/P2/P3) visualization
- Cost analysis with detailed breakdown

## Installation

Install via HACS (category: Plugin) or manually by copying the card file to your `www/community/octopus_energy_es/` directory.

## Requirements

Requires the [Octopus Energy España Integration](https://github.com/bthos/ha-octopus-energy-es) to be installed and configured.
